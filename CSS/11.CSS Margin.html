CSS margin To set the outer space of the element - Margin - Top - Margin - Right - Margin - Bottom - Margin - Left ____________________________________ | Paragraph | |__________________________________| Margin Values - Length - px, cm, pt - % - width of the element - Auto - Automatically calculate by browser - Inherit - margin values taken from its parent Negative Margin - To overlap the element Margin Collapse - If the margin values of two consecutive element is different the large marginis taken into account. - If the margin value of two consecutive element is same then one margin value is taken into account.
