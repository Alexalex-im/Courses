CSS text - color - Text -indent - Text -align - Text -decoration - Text -transform - Text -shadow - Letter -spacing - Word -spacing - Line -height - White -space CSS color It specifies the color of the text - Color name(red,blue...) - Rgb value(rgb(255, 34, 44)) - Hex value(#ffooff) CSS text-indent To set the indention of the first line of the text - Length values - px,pt,cm etc - Percentage(%) CSS text-align To set the horizontal alignment of the text - Left, Right, Center, Justify CSS line-height To set the space between the lines - Length values - Percentages CSS text-decoration It specifies the decoration of the text - Underline, overline, line-through, none CSS text-transform To set the cases for the text - Uppercase, lowercase, capitalize CSS text-shadow It adds the shadow effect to the text - Horizontal, vertical, blur with color CSS letter-spacing To set the space between each character of the text - Normal, length values, negative values CSS word-spacing To set the space between each word of the text - Normal, length values, negative values CSS white-space To set the white space of the text - Normal, pre, nowrap
