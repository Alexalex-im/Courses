Javascript-Regular Expression - It is a sequence of character that forms a search pattern - Can be a single character or complicated pattern - Used to perform all type of text search and text replace opertions RegExp-syntax - var regex = new regExp( pattern, modifiers ) - var regex = /pattern /modifiers - pattern - A string that specifies the pattern of the regex or another regex - modifiers - ( Optional ) - 'g', 'i', 'm' - /..../ - Creation of RegExp Using String Methods - search() - Uses an expression to search for a match and returns position - replace() - return a modified string where the pattern is replaced - match() - Return all matches of the pattern in the string RegEx-Patterns _______________________________________________________________ | [a b c] | Find any Character between the brackets | --------------------------------------------------------------- | [0 - 9] | Find any digits between the brackets | --------------------------------------------------------------- | [x | y] | Find any of the alternative separated with | | --------------------------------------------------------------- Meta character __________________________________________________________________________________ | \d | Digits | ---------------------------------------------------------------------------------- | \s | white Space | ---------------------------------------------------------------------------------- | \b | Match at the beginning of the word \b WORD end of the word WORD \b | ---------------------------------------------------------------------------------- RegEx-Quantifiers - Quantifiers - Quantifiers _______________________________________________________________ | x+ | Matches any string that contains atleast one x | --------------------------------------------------------------- | x* | Zero or more occurances of x | --------------------------------------------------------------- | x? | Zero or one occurances of x | --------------------------------------------------------------- - Using regExp methods : - text() - Returns true or false depending on the result - exec() - Returns found text as on object if the match is found else returns empty object
