Javascript-Strict Mode - "use strict"; - Javascript code should be executed in strict mode. - Modern browsers only support strict mode. - Included to the beginning of a script or a function will be recognized. Use of strict Mode - It makes to write secure javascript - Changes previously accepted bad syntax into real errors. Strict Mode - Using a variable without declaring is not allowed - Using an object without declaring is not allowed - Deleting a variable, an object, a function is not allowed - Duplicating parameter name is not allowed - Word eval, arguments cannot be used as variable names - "this" keyword in functions behaves differently in strict mode
