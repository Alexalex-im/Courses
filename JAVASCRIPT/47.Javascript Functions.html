Javascript-Functions - A function is a group of statements that perform specific tasks - Function is executed when calls it. Function-uses - reduce the repetition of a code within a program - Makes the code easier to maintain - Makes easier to eliminate the errors Function definition - Defined with the function keyword - Function declaration or function expression Adding parameters to function - Function parameters are the names listed in the function definition - Function arguments are the real values passed to and received by the function Function Calling - The code inside a function is not executed when it is defined - The code is executed when the function is called( invoked ) - Function can be called by many ways
