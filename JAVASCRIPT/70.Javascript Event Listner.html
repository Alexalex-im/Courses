Event Capturing / Bubbling

In bubbling ( default ) the inner most element's event is handled first and then the outer
In capturing the outer most element's event is handled first and then the inner.



Javascript - EventListener - Event listener is a procedure that wait for an event to occur. - Able to assign as many event listener to the same event for the same element - Events performed by user of the webpage, Event handler is performed by browser, Event Listener is like event handler able to assign multiple functions to same event. Event Listener - Syntax - addEventListener(event,function,useCapture) - Event - type of the event ( click, focus ) - Function - to call when the event occurs - useCapture - Boolean, optional default is false - Capturing - execute outer element event - Bubbling - execute inner element event - Remove Event Listener - removeEventListener( event, function ) - Removes event handlers that have been attached with the addEventListener method
