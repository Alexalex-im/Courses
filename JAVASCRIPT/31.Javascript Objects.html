Javascript-Object - Javascript is an Object Oriented Programming (OOP) language. - Object belong to Non-primitive datatype. - Object are used to store collections of data and complex entities. - Object are composed of properties and methods. Object creation - By object literal var objname = {prop1:value1, prop2:value2...propN:valueN} - By creating instance of object - new keyword var objname = new Object(); - By using and object constructors - new keyword To create a function with arguments Accessing ways ways to access the properties and methods of objects - By dot operator - objname.property - Square brackets - objname[property]
