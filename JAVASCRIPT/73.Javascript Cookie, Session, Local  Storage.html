Local Storage / Session Storage / cookie

Javascript Storage methods are explained with examples. In session and local storage data are stored by browser. In cookie, data are stored and used by both server & client.




Javascript Cookies / session / Local Storage - session - Used to store untill the current browser tab exists - Local storage - Used to store data permanently in hard drive - Cookies - a piece of data that stored in our system as a text file to be accessed by our browser.
