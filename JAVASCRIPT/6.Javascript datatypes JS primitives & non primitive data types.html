Javascript-datatypes - Primitive data types - Non primitive data types Primitive data types - String // "" or '' or `` - Number // number or float value or exponential or notanumber NaN or infinity or -infinity - Boolean // true or false or 0 or 1 - Undefined // novalue or null - Null // undefined or object Non-primitive data types - Object - Array - Date The type of operator - To find the data type of a variable - String - return as string - Number - return as number - Boolean - return as boolean - Nan - return as number - Null, object, date(), array - return as object - Function - return as function - Undefined - return as undefined
