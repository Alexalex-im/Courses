Javascript Function - bind(), call(), apply() - bind() - Binds / attach an object to a function - Reference it using "this" - To execute function, need to call the function - apply() - Allows to call a function by passing the arguments in a single array - call() - Allow to call a function by passing arguments list Syntax - call( this, arg1, arg2, ...) - apply( this, [args Array]) - bind ( this, arg1, arg2 ...) Difference - bind() - Use to call function later - Useful in events, callback functions - return a function - Call() - Execute function immediately - arguments list - Apply - Execute function immediately - arguments array
