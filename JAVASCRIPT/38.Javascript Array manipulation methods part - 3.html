Array methods - forEach() - To run a function for every element - reduce() - Used to calculate single value based on array. - isArray() - checks for being an array - map() - It calls the specified function for every element into new array - copywithin() - Copies the part of the given array to new one - some() - Check atleast one element in the array satisfying the condition (Boolean) - every() - To check all the element in an array satisfying the condition (Boolean) - filter() - Return an array of all matching elements - includes() - It checks whether an array contains the specified element - sort() - It return the element in the array in a sorted order.
